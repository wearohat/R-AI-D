# Installation

## Clone Repository

```bash
git clone https://github.com/wearohat/R-AI-D.git
```

---

# Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Install Ollama

Official website:
https://ollama.com

---

# Pull A Model

Example:

```bash
ollama pull llama3.2:3b
```

---

# Verify Ollama

```bash
ollama list
```

---

# Run Project

```bash
python main.py example_target.com
```
